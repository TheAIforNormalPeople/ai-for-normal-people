## üé≠ CHARACTER VOICE ROTATION SYSTEM

### **Kai Sounds (3-4 different per episode):**
**Never repeat same combo from last episode**

**Sound Bank:**
- BEEP (baseline, neutral)
- WHIRR-CLICK (processing uncertainty)
- CHK-CHK (stress, determination)
- DING (realization)
- soft chime (warmth, affection)
- ALARM-BUZZ (panic)
- whistle (curiosity)
- mechanical purr (contentment)
- rhythmic ticking (anticipation)
- processing hum
- alert chime
- systems check
- gentle beep
- analysis tone
- soft whirr
- data ping
- scanner sweep
- monitoring pulse
- detection click
- frustrated pulse
- confused whirr
- irritated whirr

**Monitoring Phrases (Vary):**
- "Analysis suggests..."
- "Detection patterns show..."
- "My sensors indicate..."
- "Monitoring reveals..."
- "Alert: [observation]"
- "Pattern detected:"
- "System analysis:"
- "Scanning indicates:"
- "Data shows:"
- **NEVER:** "Probability that..." more than once per episode

### **Vector Phrases:**
**Never repeat from last episode**

**Enthusiasm Alternatives:**
- "FASCINATING!" (max 1 per episode)
- "Wait, this is interesting!"
- "OH! That's actually..."
- "This is really cool because..."
- "You know what's wild about this?"
- "Here's the thing that gets me excited..."
- "This is where it gets interesting..."

**Self-Correction Types (Vary):**
- "Wait, let me rephrase..."
- "Actually, that's not quite right..."
- "Hang on, I need to think about this..."
- "Let me adjust that..."
- "Actually, scratch that..."

### **Recurse Actions:**
**Never repeat from last episode**

## üé® FORMATTING STANDARDS

### **Dialogue:**
- Use `{{< dialogue char="CharacterName" >}}` shortcodes
- Close with `{{< /dialogue >}}`
- Include stage directions in italics: `*action*`
- Human dialogue: `**[Human]:** *stage directions*`

### **Bounce Dialogue:**
- Use `<div class="dialogue-box bounce">` for his box
- Or regular dialogue initially, then box appears
- Decision made per episode

### **Meanwhile Scenes:**
- Use `<div class="meanwhile-scene">` wrapper
- Only when needed (not every episode)
- Keep to 180-200 words
- End with `**[Back with the group...]**`

### **Sections:**
- Use `##` for main sections
- Use `###` for subsections
- "Key Takeaways" as `## Key Takeaways`
- "Sources & Further Reading" as `## Sources & Further Reading`
- "What's Next?" as `## What's Next?`
