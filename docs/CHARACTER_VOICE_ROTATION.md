## BANTER PATTERNS

### Interruption Patterns
1. Character A talking → Character B interrupts mid-sentence
2. Character A explaining → Character B questions → Character A adjusts
3. Character A makes claim → Character B challenges → Character A defends
4. Character A teaching → Character B adds detail → Character A acknowledges
5. Character A excited → Character B calms → Character A continues

### Comedy Moments
1. Equipment malfunction during serious moment
2. Character misunderstands human behavior
3. Meta-commentary about being AI
4. Inside joke callback
5. Physical comedy (tripping, dropping, etc.)
6. Character contradicts themselves
7. Character realizes mistake mid-sentence
8. Environmental detail interrupts dialogue
