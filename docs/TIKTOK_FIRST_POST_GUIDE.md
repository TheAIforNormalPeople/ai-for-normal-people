## üì± Step 1: Create TikTok Account (5 Minutes)

### On Phone (Easiest):

1. **Download TikTok app**
   - Go to App Store (iPhone) or Google Play (Android)
   - Search "TikTok"
   - Tap "Install"
   - Wait for download

2. **Open TikTok app**
   - Tap TikTok icon
   - App opens

3. **Sign Up**
   - Tap "Sign Up" (bottom)
   - Choose "Use email" or "Use phone"
   - Enter email/phone
   - Enter password
   - Tap "Sign Up"
   - Done!

4. **Choose Username**
   - Type username (e.g., "thenormalpeople" or "ai4normalpeople")
   - If taken, try variations (add numbers or underscores)
   - Tap "Next"
   - Done!

**That's it. You have a TikTok account!**

---

### On Computer (Alternative):

1. **Go to tiktok.com**
   - Open web browser
   - Go to tiktok.com
   - Click "Sign Up" (top right)

2. **Sign Up**
   - Enter email
   - Enter password
   - Click "Sign Up"
   - Verify email (check inbox)
   - Done!

3. **Choose Username**
   - Type username
   - If taken, try variations
   - Click "Next"
   - Done!

**Note:** Phone is easier for posting videos, but computer works too!

## ‚öôÔ∏è Step 2: Switch to Creator Account (2 Minutes)

**Why:** Better analytics, access to Creator Fund at 10k followers

**Steps:**

1. **Open TikTok**
   - Tap/click "Profile" (bottom right)

2. **Open Settings**
   - Tap "..." (three dots, top right)
   - Tap "Settings and privacy"

3. **Switch Account Type**
   - Tap "Account"
   - Tap "Switch to Creator Account"
   - Follow prompts (choose category: "Education" or "Tech")
   - Tap "Switch"
   - Done!

**That's it. You're now a Creator Account!**

## üé¨ Step 3: Create Your First Video (30-60 Minutes)

### Option A: Simple Text + Image Video (Easiest - 30 Minutes)

**Tools:**
- **CapCut** (free, mobile/desktop) - Video editing
- **Canva** (free, web) - Image creation
- **Free TTS** - Text-to-speech (Google TTS, Edge TTS)

**Steps:**

1. **Get Content from n8n**
   - Run n8n workflow (generates TikTok description/script)
   - Copy description and script
   - Save to notes

2. **Create Images in Canva**
   - Go to canva.com
   - Sign up (free)
   - Create new design: "Video" ‚Üí "TikTok" (1080x1920)
   - Add text overlay (episode title)
   - Add character art (if you have it)
   - Export as image (PNG)
   - Create 3-5 slides (one per key point)
   - Export all images

3. **Create Video in CapCut**
   - Open CapCut app
   - Tap "+ New Project"
   - Select images (all slides)
   - Set duration (3-5 seconds per slide)
   - Add transitions (optional)
   - Add text (optional)
   - Export video
   - Done!

4. **Add Voice-Over (Optional)**
   - Use free TTS (Google TTS, Edge TTS)
   - Generate voice-over from script
   - Import to CapCut
   - Sync with images
   - Export video
   - Done!

**Time:** 30-45 minutes (first time), 20-30 minutes (subsequent)

---

### Option B: Character Art + Text Overlays (Recommended - 45 Minutes)

**Tools:**
- Same as Option A
- Plus: Character art generation (you have prompts!)

**Steps:**

1. **Generate Character Art** (5 minutes)

2. **Create Video in CapCut** (20 minutes)

3. **Add Voice-Over** (15 minutes)

   - Generate voice-over from script
   - Import to CapCut
   - Sync with video
   - Export
   - Done!

**Time:** 45-60 minutes (first time), 30-45 minutes (subsequent)

## üì§ Step 4: Post Your Video (2 Minutes)

**Steps:**

1. **Open TikTok**
   - Tap "+" button (bottom center, big plus icon)
   - Or tap "Upload" (if on computer)

2. **Choose Video**
   - Tap "Upload" (choose video file)
   - Select your video
   - Wait for upload

3. **Add Description**
   - Paste description from n8n (or write your own)
   - Include hashtags (n8n includes these!)
   - Example: "Episode 29: How to Work With AI Systems You Can't Fully Control. Full episode link in bio! #AI #MachineLearning #TechEducation #LearnAI"

4. **Add Hashtags** (n8n includes these!)
   - #AI
   - #MachineLearning
   - #TechEducation
   - #LearnAI
   - #AITutorial
   - #TechTips

5. **Post**
   - Tap "Post" (bottom right)
   - Wait for upload (30-60 seconds)
   - Done!

**That's it. You just posted to TikTok!**
