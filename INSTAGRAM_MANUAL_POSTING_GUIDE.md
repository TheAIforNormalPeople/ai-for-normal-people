## ğŸ“± Posting Steps

### Step 1: Create Image
1. Generate episode artwork or character art
2. Add episode title (if needed)
3. Save as 1080x1080px

### Step 2: Write Caption
1. Use template above
2. Customize for episode
3. Add relevant hashtags (5-10 max)

### Step 3: Post to Instagram
1. Open Instagram app
2. Tap "+" to create new post
3. Select image
4. Add caption
5. Add location (optional)
6. Tag account (if applicable)
7. Post!

### Step 4: Add to Story
1. After posting, tap "Share to Story"
2. Add character sticker or quote
3. Add link sticker (if you have 10k+ followers)
4. Or add "Link in bio" text
