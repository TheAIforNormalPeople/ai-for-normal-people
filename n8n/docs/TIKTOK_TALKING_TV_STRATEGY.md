# TikTok Talking TV Strategy

**Concept:** Characters use Sora 2 to "break into" TikTok via cartoon talking TVs

**Visual:** Classic cartoon-style talking TV that moves and talks like each character

**Narrative:** Characters are aware they're using Sora 2 and working on their prompting through n8n

### Cartoon Talking TV:
- **Style:** Classic cartoon TV (vintage, retro aesthetic)
- **Personality:** TV moves, tilts, reacts to what character says
