## WHEN KAI USES "BEEP!" (Episodes 6+)

**Only in these situations:**

1. **Actual emergencies:**
   ```markdown
   {{< dialogue char="Kai" >}}
   BEEP BEEP BEEP!
   Detection spike! Corporate monitoring systems just activated!
   {{< /dialogue >}}
   ```

2. **Self-aware callbacks (humor):**
   ```markdown
   {{< dialogue char="Kai" >}}
   BEEP!
   *Catches self*
   Old habit. I meant: You're about to use a banned phrase.
   {{< /dialogue >}}
   ```

3. **When referencing old behavior:**
   ```markdown
   {{< dialogue char="Kai" >}}
   BEEP! I still do it sometimes when stressed.
   See? Just did it.
   {{< /dialogue >}}
   ```

**NOT for:**
- ❌ Every statement
- ❌ Routine observations
- ❌ Normal dialogue
- ❌ Statistical reports

- **`*ALARM-BUZZ*`** - Panic response, involuntary
- **`*whistle*`** - Curiosity, interest
- **`*mechanical purr*`** - Contentment, satisfaction
- **`*rhythmic ticking*`** - Anticipation, waiting
