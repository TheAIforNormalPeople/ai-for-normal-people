---

## üìã Review Process

### Step 1: Identify All Made-Up Data

**Search for patterns:**
- `%` (percentages)
- `statistic`, `data`, `study`, `research`
- `analysis:`, `pattern detected:`, `alert:`
- Numbers followed by `%` or `percent`

### Step 2: Categorize Each Instance

**Category A: Essential to Story**
- Statistics that are central to the educational point
- **Action:** Find real source OR acknowledge uncertainty

**Category B: Character Voice**
- Statistics that show character personality (Kai's detection)
- **Action:** Keep but make it clear it's character speculation/estimation

**Category C: Unnecessary**
- Statistics that don't add value
- **Action:** Remove entirely

### Step 3: Fix Each Episode

**Option 1: Find Real Source**
- Research actual studies/data
- Cite properly
- Add to episode notes

**Option 2: Acknowledge Uncertainty**
- Change: "15-20% retention" 
- To: "Studies suggest passive learning has lower retention, but exact numbers vary"
- Or: "I don't have exact data, but the pattern suggests..."

**Option 3: Character Speculation**
- Keep statistics but make it clear it's Kai's estimation
- Add: "Based on my analysis of available data..."
- Or: "My calculations suggest approximately..."

**Option 4: Remove**
- If not essential, just remove the statistic
- Keep the educational point without the number
